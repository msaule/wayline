Privacy

Wayline uses the trip details you enter to build a route and return the finished itinerary.

The current product does not include user accounts or a built-in trip library. Exports such as PDF, text, and JSON are generated for you to save where you want.

Route and place lookups use Google Maps Platform services. Those requests are made by the app so the planning result can be built and returned to you.
